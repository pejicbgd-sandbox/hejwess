Easy to use text slider widget.

== Description ==

WP Text Slider Widget help you showcase your News, content, important notices on the website.

* Show scrolling text on any page using sidebar
* Can be used as advertising, notice, important updates
* Compatibility with many other plugins

= Features =

* Easy to use.
* Cross browser compatible (IE9+, Safari, Chrome, Firefox).

== Installation ==

1. Upload the `wp-text-slider-widget` folder to the `/wp-content/plugins/` directory or install it from the plugin directory via your Plugins dash.
2. Activate the Text Slider plugin through the 'Plugins' menu in WordPress
3. Add your texts the plugin by going to the `Wp Text Slider Widget` tab in your admin menu.
4. Visit your Widgets Section ( Apperance > Widgets *Wp Text Slider*)
5. Drag widget in the sidebar.

== Frequently Asked Questions ==

= Can I define the my own css for text slider widget? =

Yes, you can define the your own css. Open the "wp-content/plugins/wp-text-slider-widget/css/style.css" file then write your own css.
